## Overview

**Lernify X** is a simple web-based **Learning Management System (LMS)** that allows teachers to create courses and assignments while students can enroll in courses, submit assignments, and track their progress.

The project is built as a **single-page web application** using **HTML, TailwindCSS, and JavaScript**, with **LocalStorage used as the data storage**. This allows the application to run directly in the browser without requiring a backend server.

## Features

### Authentication

* User registration and login
* Role selection (Student / Teacher)

### Teacher Features

* Create and manage courses
* Add assignments to courses
* View student submissions
* Grade assignments
* Upload course materials

### Student Features

* View available courses
* Enroll in courses
* Submit assignments
* View grades and feedback
* Participate in discussion forums

### Additional Features

* Notification system
* Basic plagiarism detection simulation
* Assignment deadline alerts
* Student submission streak system

## Technologies Used

* **HTML5** – Structure of the application
* **TailwindCSS** – UI styling
* **JavaScript (Vanilla JS)** – Application logic
* **LocalStorage** – Client-side data storage

## How It Works

The application runs entirely in the browser.

All data such as:

* users
* courses
* assignments
* submissions
* grades

are stored in **LocalStorage**, allowing the LMS to function without a backend.
